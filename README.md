Overview
This application streamlines report management for local accounting firms, enabling them to track report submissions, manage payments, and send customer notifications efficiently.

Features
Report Tracking: Monitor which businesses have submitted reports.
Payment Management: Track payments and follow up on outstanding balances.
Customer Notifications: Automatically notify customers about report submissions and payment reminders.
User-Friendly Interface: Simple navigation for accounting staff.
and more ...
Technologies Used
Alpine.js, Laravel (with Livewire)
